You are alone in the woods with only your ears to guide you. Listen closely to your surroundings as you make your way through this unique audio-based experience! Discover the secrets of the Echospace Forest and track down the mysterious radio transmissions. Maybe you will even make it out alive...

Controls: 
If using a controller (preferred): Use the left stick to walk and the right stick to turn
If using mouse and keyboard: Use WASD to walk and move the mouse to turn

Additional controls:
Use 'C' to change the overhead view
Use 'X' to reset the game
Use 'escape' to exit the game
